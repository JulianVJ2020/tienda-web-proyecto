Acerca de Nosotros

Somos una tienda virtual creada como parte del proyecto académico del módulo de Desarrollo Web. Nuestro objetivo es simular un sistema de comercio electrónico sencillo, funcional y accesible.

Quiénes Somos

Somos un proyecto individual enfocado en el diseño y desarrollo de un sitio web tipo e-commerce, aplicando buenas prácticas de programación, estructura HTML, diseño con CSS y componentes responsivos utilizando Bootstrap. Buscamos demostrar las bases de una plataforma de ventas moderna.

Misión

Crear una plataforma digital organizada, intuitiva y visualmente atractiva que permita mostrar productos de manera clara y ordenada, demostrando habilidades de maquetación, diseño y desarrollo web responsivo.

Visión

Consolidar este proyecto como una base sólida para futuros desarrollos web, integrando elementos avanzados de accesibilidad, interacción dinámica y funcionalidades que reflejen el crecimiento en el aprendizaje del estudiante.
